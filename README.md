## Useful Cribl Stream CLI Commands:
- Display a list of all available commands:
```
./cribl help -a
```
- Start Crible Stream Service
```
./cribl start
```
- Stop Crible Stream Service
```
./cribl stop
```
- Start Crible Stream Service
```
./cribl restart
```
- Check Cribl Status
```
./cribl status
```
